# Celebrity Chooser

#### This is a web app that matches you to a celebrity based on a few pieces of criteria.

#### By Scott McIntire and Danny Pineda

## Description

A user answers a few questions about themselves and then gets matched to a celebrity.

## Installation instructions

Clone this repository and run the index page in a web browser.

## Technologies Used

HTML, CSS, JavaScript, JQuery, and Bootstrap
